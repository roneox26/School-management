## 🎨 Visual Hierarchy

### **Typography Levels**
```
H1: School Name (1.75rem - Bold)
H2: Section Titles (1.5rem - Bold)
H3: Card Values (2rem - Bold)
H4: Card Titles (1.25rem - Semi-bold)
P:  Descriptions (0.95rem - Normal)
```

### **Color Usage**
```
Primary Text:     #1f2937 (Near Black)
Secondary Text:   #4b5563 (Dark Gray)
Tertiary Text:    #64748b (Medium Gray)
Muted Text:       #9ca3af (Light Gray)

Highlights:       Pastel Color Palette
Accents:          Blue Gradient
```

## ⚡ Animation Effects

### **Entrance Animations**
- **Slide Up**: 350ms ease-out (Staggered delay)
- **Fade In**: 250ms ease-out

### **Interaction Animations**
- **Hover Lift**: Transform -4px, box-shadow increase
- **Button Press**: Scale 0.98
- **Focus Glow**: Colored ring effect

### **Timing Functions**
```
Fast:     150ms
Base:     250ms
Slow:     350ms
Easing:   cubic-bezier(0.4, 0, 0.2, 1)
```
