## How to Use

### In Templates
```html
<!-- Show success message -->
<div class="alert-custom alert-success">
    Operation completed successfully!
</div>

<!-- Show error message -->
<div class="alert-custom alert-error">
    Something went wrong. Please try again.
</div>

<!-- Empty state -->
<div class="empty-state">
    <div class="empty-state-icon">
        <i class="fas fa-inbox"></i>
    </div>
    <div class="empty-state-title">No Items</div>
    <a href="/add" class="empty-state-action">Add Item</a>
</div>
```

### In Python
```python
# Use safe database functions
students = safe_query_db('student', is_active=True)

result = safe_save_to_db('student', data)
updated = safe_update_in_db('student', id, data)
deleted = safe_delete_from_db('student', id)

# They all handle errors automatically
# Errors are logged to console
# Operations fail gracefully
```
